# CRM
Customer Relationship Management App

This is an application where you can-
* View Regions
* View Customers
* Edit Customers
* Add a new Customer
* Clear Data
* Set a reminder to contact a customer

## Install the dependencies

run-
```
yarn
```

## Start the application

run-
```
yarn start
```
