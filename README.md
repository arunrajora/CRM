# CRM
Customer Relationship Management App

This is an application where you can-
* View Regions
* View Customers
* Edit Customers
* Add a new Customer
* Clear Data
* Set a reminder to contact a customer

## Install the dependencies

run-
```
npm install
```
or
```
yarn
```

## Start the application

run-
```
npm start
```
or
```
yarn start
```
